2. Make connections
Access Point-PT to Switch x2 (Auto Choose)

WSN P6 Create a basic MANET implementation simulation for packet animation and packet trace.
Steps:
1. Adding the devices:
3 1841 Routers
3 Switch-PTs
3 AccessPoint-PTs
2 Laptops
2 PC-PTs
2 Smartphones

2. Add serial ports in the router
Pysical - turn off - WIC-2T - turn on

3. Adding Connections
Access Point - Switch - Router: Auto Choose
Router0 - Router1: Copper Cross Over
Router1 - Router2: Auto Choose

4. Configure Routers:
Router0:
Config - Interface - FastEthernet0/0 - Port Status: on - IPv4: 10.0.0.1
FastEthernet0/1 - Port Status: on - IPv4: 192.168.1.1
RIP - Add : 10.0.0.0, 192.168.1.0, 192.168.2.0, 192.168.3.0

Router1:
Config - Interface - FastEthernet0/0 - Port Status: on - IPv4: 10.0.0.2
FastEthernet0/1 - Port Status: on - IPv4: 192.168.2.1
Serial0/1/0 - Port Status: on - Clock Rate: 1200 - IPv4: 192.168.4.1
RIP - Add : 10.0.0.0, 192.168.1.0, 192.168.2.0, 192.168.3.0, 192.168.4.0

Router2
Config - Interface - FastEthernet0/0 - Port Status: on - IPv4: 10.0.0.3
FastEthernet0/1 - Port Status: on - IPv4: 192.168.3.1
Serial0/1/0 - Port Status: on - IPv4: 192.168.4.2
RIP - Add : 10.0.0.0, 192.168.1.0, 192.168.2.0, 192.168.3.0, 192.168.4.0

5. Configure the Access Points
Access Point0/1/2
Config - Interface - Port 1 -  Port Status: on - SSID: cs1/2/3 - Authentication - WEP - WEP Key: 0123456789

6. Configure the devices
Laptop0/1
turn off - remove port - drag WPC300N port - turn on
Config - Interface - Wireless0 - Port Status: on - SSID: cs1 - Authentication - WEP - WEP Key: 0123456789 - IP Configuration - Static - IPv4: 192.168.1.2/3

PC0/1
turn off - remove port - drag WPC300N port - turn on
Config - Interface - Wireless0 - Port Status: on - SSID: cs2 - Authentication - WEP - WEP Key: 0123456789 - IP Configuration - Static - IPv4: 192.168.2.2/3

Smartphone0/1
Config - Interface - Wireless0 - Port Status: on - SSID: cs3 - Authentication - WEP - WEP Key: 0123456789 - IP Configuration - Static - IPv4: 192.168.3.2/3

8. Ping devices to confirm connections

WSN P8 Create MAC Protocol simulation implementation for Wireless Sensor Network.
Steps:
1. Add Devices
1 HomeRouter-PT-AC
3 PC-PTs
3 Laptops

2. Configure the devices
turn off - remove port - drag WPC300N port - turn on

3. Check connections

4. Configure the wireless router by adding the MAC Addresses in the MAC Filter
GUI - Wireless - Wireless MAC Filter - Enabled - Permite PCs listed below to access wireless network - Add All the MAC Addresses - Save Setting (scroll down)
For Adding MAC Address:
PCs/Laptops Config - Interface - Wireless0 - Copy the MAC Address
Paste it in the GUI Window (ex: 0001.9624.9D6E -> 00:01:96:24:9D:6E) 

5. Ping Devices to check if its working.
Pinging PC0 from Laptop0
On PC0:
CMD - ipconfig - copy the IPv4 address
On Laptop0:
CMD - Ping (the copied IPv4 address of PC0)



WSN P9 Create a mobile network using Cell Tower, Central Office Server, Web Browser and Web server, Simulate the connections between them.
Steps
1. Adding Devices
1 HomeRouter-PT-AC
1 Hub-PT
1 1841 Router
1 Smartphone
1 Cel-Tower
1 Central Office Server
1 Server-PT

2. Connections
HomeRouter - Hub (Copper Cross Over Internet - FastEthernet0)
Hub - 1841 Router (Auto Choose)
1841 - Server-PT (Auto Choose)
Hub - Central-Office-Server (Auto Choose)
Cel-Tower - Central-Office-Server (Auto Choose)

3. Turn on the port status for 1841 router
FastEthernet0/0 and FastEthernet1/0

4. Add the static IP in the home router
Wireless Router0 - Config - Interface - Internet - Static - IPv4 Address: 20.0.0.1 - Default Gateway: 20.0.0.2

5. Add the IP Adresses in the 1841 router
Router0 - Config - Interface - FastEthernet0/0 - IPv4: 20.0.0.2 (Default Gateway)
FastEthernet1/0 - IPv4: 21.0.0.1
Router0 - Config - Routing - RIP - Network: ADD (20.0.0.0, 21.0.0.0, 22.0.0.0)

6. Add IP for the server
Server0 - Config - Interface - FastEthernet0/0 - IPv4: 21.0.0.2 
Server0 - Config - Global - Default Gateway: 20.0.0.2

7. Send Message from smartphone to server
Click the Mail Icon - Selct Smartphone - Select Server0
